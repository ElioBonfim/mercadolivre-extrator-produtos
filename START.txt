🚀 COMO INSTALAR
═══════════════════════════════════════════════════════════════════

OPÇÃO 1: Automático (macOS)
───────────────────────────
1. Dê dois cliques em: INSTALAR.command
2. O Chrome abrirá em chrome://extensions/
3. Siga as instruções na tela


OPÇÃO 2: Manual (Qualquer Sistema)
───────────────────────────────────
1. Abra o Chrome
2. Digite na barra: chrome://extensions/
3. Ative "Modo do desenvolvedor" (canto superior direito)
4. Clique "Carregar extensão sem empacotamento"
5. Selecione ESTA PASTA (MercadoLivre-Extension)
6. ✅ Pronto!


📖 COMO USAR
═══════════════════════════════════════════════════════════════════

1. Acesse: mercadolivre.com.br
2. Faça uma busca (qualquer produto)
3. Clique no ícone 🛒 na barra do Chrome
4. A lista de produtos aparecerá automaticamente

🆘 PROBLEMAS?
═══════════════════════════════════════════════════════════════════

❌ "Nenhum produto encontrado"
   → Você está em uma página vazia
   → Tente fazer uma busca no Mercado Livre

❌ Extensão não aparece
   → Verifique se "Modo do Desenvolvedor" está ON
   → Recarregue a página (F5)

❌ Alguns dados faltam
   → Mercado Livre muda a estrutura
   → Clique "↻ Extrair" novamente
